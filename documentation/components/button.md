# Button


## Usage
```pug
+t3-button(
  t3-button-type="secondary"
) Button text
```


## Pug Attributes

### t3-button-type

Sets the style of the button

**Available options:**
- primary (default)
- secondary


## SCSS Variables

```sass
$t3-button-border-radius: $t3-border-radius;
$t3-button-font-size: $t3-font-size-default;
```
